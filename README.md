# test-maven-plugin

This is a basic test of the creation of a Maven Plugin


### Prerequisites
The following items should be installed in your system:
* Maven 3 (http://www.sonatype.com/books/mvnref-book/reference/installation.html)
* git command line tool (https://help.github.com/articles/set-up-git)
* Eclipse with the m2e plugin or STS (m2e is installed by default in STS.  http://www.springsource.org/sts)

### Steps:

1) In the command line
```
git clone https://github.com/jorgevs/test-maven-plugin.git
```
2) Inside Eclipse
```
File -> Import -> Maven -> Existing Maven project
```
